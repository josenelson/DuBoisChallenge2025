## Installation and Setup Instructions

Clone down this repository. You will need `node` and `npm` installed globally on your machine - currently tested with v20.18.2.  

Installation:

`npm install`  

To Run Locally:  

`npm run start`  

and navigate to http://localhost:9500/

To Build:

`npm run build`
